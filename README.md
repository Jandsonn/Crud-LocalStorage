# Crud-LocalStorage
Este é um Crud Simples utilizando a tecnologia Javascript puro, com armazenamento dos dados no localStorage
Esse projeto tem um layout simples, mas que permite cadastrar e fazer a edição dos produtos. 
O armazenamento deste projeto utiliza-se apenas o próprio local do Navegador, permitindo os dados até quando ao reiniciar o computador.
Não persiste permanentemente os dados mas a ideia era só fazer uma conexão no LocalStorage para fins de estudos Javascript.
